 # 🚀 github_badge
 > Débloquer des badges GitHub rapidement

- Ce repo est conçu pour débloquer plusieurs badges GitHub en quelques minutes.

## Badges débloqués
- ⭐ Starstruck
- 🐛 Issue Opened
- 🔧 Pull Request Merged
- 🦈 Pull Shark
- 🌐 Pages Publisher
- 🤖 Actions Contributor
- 🛡️ Dependabot Contributor
- 🟣 YOLO

## Comment l’utiliser
1. Fork ou clone ce repo  
2. Active GitHub Pages  
3. Merge la PR interne  
4. Ouvre une issue  
5. Laisse Dependabot créer une PR  
6. Merge-la  
7. Profite des badges
